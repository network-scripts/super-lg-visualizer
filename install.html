☀️ BGP AS-Path Sunburst Bookmarklet

Drag the button below into your bookmarks bar. Visit bgp.tools/super-lg, enter any prefix, then click the bookmark. Fully self-contained — no external libraries, no network requests.

Features

  • Origin AS at centre; each ring = one more transit hop outward
  • Arc width = number of paths flowing through that segment
  • AS number label drawn in every segment where it physically fits
  • Hover for org name, path count, depth, and BGP communities
  • Click any arc to zoom into that subtree; click centre to zoom out
  • Fills the entire browser window; adapts to resize
  • Repeated ASNs within one path collapsed to single hop
☀️ BGP Sunburst

Installation

  1. Show bookmarks bar: View → Toolbars → Bookmarks Toolbar
  2. Drag the green button above onto the bar
  3. Go to https://bgp.tools/super-lg and search a prefix
  4. Click the BGP Sunburst bookmark — overlay opens instantly
  5. Click ✕ (top-right) to close
